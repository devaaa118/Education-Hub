Education Hub is a Jakarta Servlet web application for managing educational resources, quizzes, and tutoring sessions. It enables teachers to upload and share materials, create and evaluate quizzes, schedule tutoring, and track student progress in a centralized, web-based interface.

Features
Teacher resource library with file uploads and external URL links
Quiz creation, assignment, and results viewing
Tutoring session scheduling and management
Student progress tracking and resource completion status
Role-based access for teachers and students
File storage and retrieval utilities for uploaded resources
